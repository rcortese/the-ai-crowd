# The AI Crowd

A homelab AI workbench for Claude-led, multi-model development workflows.

## What it is

The AI Crowd is a personal project centered on a shared AI workbench: a single environment where Claude acts as the primary orchestrator and other model CLIs can be used as delegated workers or directly when needed.

## Goals

- Keep the workflow simple and local-first
- Support multi-model development in one consistent workspace
- Preserve persistent state and project context
- Stay practical for personal homelab use

## Scope

This repository documents the project architecture and implementation direction for the workbench itself.
